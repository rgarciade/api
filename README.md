### Installation
Requires [Node.js](https://nodejs.org/) v3.5.2+ to run.

Requires [mongo](https://docs.mongodb.com/manual/administration/install-community/) server runing in localhost:27017
Install and start the server.

```sh
$ npm install 
$ npm start
```
